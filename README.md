## Usage
### Installation
```
[~/MQCA] mkdir build && cd build
[~/MCQA/build] cmake ..
[~/MCQA/build] make -j
```

### Run
```
[~/MCQA/tool] python3 run.py [circuit_index]
```
or
```
[~/MCQA/tool] python3 run.py all
```


If you want to know more about "run.py", put this command `python3 run.py help'
